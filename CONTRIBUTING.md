# Contributing to cc-worktree-squad

Thanks for helping improve this project.

## Development setup

```bash
git clone <repo-url>
cd cc-worktree-squad
npm install
npm test
```

## Branching

- Use short, descriptive branch names.
- Keep pull requests focused on one change.
- Write clear commit messages.

## Pull request checklist

- [ ] Tests pass (`npm test`)
- [ ] CLI help output still works (`node bin/ccws.js --help`)
- [ ] README/docs updated if behavior changed
- [ ] No unrelated refactors included

## Reporting bugs

Please include:

- OS and Node.js version
- Exact command used
- Expected behavior
- Actual behavior
- Error output (if any)
